# Core Model

GL has a structure.

It is not linear.
It expands and contracts with human state.

It is composed of multiple layers.

Not stacked,
but connected.

Not fixed,
but moving.

Each layer influences the others.

They form a breathing structure.

---

The loop moves through them.

Inner, outer, relational, social.

And returns again.

---

This is not a system to control.

It is a structure that changes through relation.
